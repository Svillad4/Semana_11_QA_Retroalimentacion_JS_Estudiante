# Checklist de entrega

Marca cada punto antes de subir el ZIP final a Moodle.

## Proyecto

- [. ] Abrí el proyecto en Visual Studio Code.
- [. ] Ejecuté `proyecto_a_revisar/index.html`.
- [ .] Revisé la consola del navegador.
- [. ] Ejecuté `proyecto_a_revisar/tests.html`.
- [. ] Corregí los bugs priorizados.
- [. ] Probé nuevamente después de corregir.

## Documentación

- [ ] Completé `04_registro_bugs_priorizado.md`.
- [ .] Completé `05_correcciones_aplicadas.md`.
- [. ] Completé `06_plan_mejora.md`.
- [ ] Completé `07_formato_coevaluacion.md`.
- [ .] Completé `08_matriz_pruebas_regresion.md`.

## Evidencias

- [ ] Guardé captura de consola con errores iniciales.
- [ .] Guardé captura del sistema funcionando después de corregir.
- [ ] Guardé captura de `tests.html`.
- [ ] Guardé evidencia de al menos una validación inválida.
- [ .] Guardé evidencia de un registro válido.

## Entrega

- [ .] Revisé que la carpeta mantenga su estructura.
- [ ] Comprimí la carpeta completa en ZIP.
- [ ] Subí el ZIP a Moodle.
- [. ] Verifiqué que el archivo subido sea el correcto.
